HCI - Spring 2021 - Project 1 Report

1. Motivation


Design plays an essential role in communicating between objects and users. Any device should be intuitive and easy to use. That being said, some give the user a tough time figuring out how they work. In this project, we will look into the microwave, a ubiquitous device we see every day, to see its current design and make it more user-friendly.



2. Current Desgin

I bought a new microwave from Amazon a few months ago. It was a COMFEE's countertop microwave, first released in 2018. Here is a photo of the microwave with the control interface on the right.




3. Key Features

To see if the interface makes doing those common things easier, first, we need to identify common uses and rare uses of a microwave.

Common use:

  • Express cooking: For example, pressing on "1" to heat the food up in 1 minute.

  • Set up both time cook and power level: Let's say we want to heat the food up in 1 minute with 80% power: either using a combination of "Time", 0", "1", "0", "0", "Power", "8", "start" or using express cook by pressing "1", and then changing the power level while heating up using "power", "8".

Rare uses: Other buttons like defrost, beverage, vegetables



4. Potential Issues

What are the potential issues with this design?

  • Too many buttons, most of them are rarely used
  • Buttons are not tactile
  • It takes many steps to set the power level, not to mention that the power setting is not intuitive as the user needs to choose the time cook first.


The microwave supports the user figure out by some user instructions on the keypad. It provides feedback to the user using sounds and the LED screen.


This current design may cause some trouble for the user, such as it is hard to figure out how to set up the power level and not sure which level is the max/min. Besides, quite many steps for setting up the power level.



5. Suggestions

It should easy to see these two primary features (Time Cook, Power Level) and require minimum steps to proceed.

  • Make the power level more visible with a knob. Its values are ranging from 1 (least power) to 10 (most power). The user can easily adjust the power level and can see what level they are using easily.
  • For time cook, using "adding" rather than "setting". It will "add" an amount of time to the current time left when a button is pressed. For example, pressing on "+6" twice will give us 12 minutes. It can also be archived by other combinations, such as "+7" and "+5", as long as they add up to 12. By doing this, we do not need to go through so many steps to set up the time as before (i.e., "Time", "1", "2", "0", "0", "START").
  • Some other suggestions would be elaborated in the next section.



Demo

In this project, we use Processing to demonstrate the new design.
In addition to nine number buttons ("+1" to "+9"), the microwave has a "STOP" button and a "+30s" button. To start, the user can press any "+" button. It will add that amount of time to the current time and start the microwave if it is not running. There is also a knob to control the power level. The brightness of the microwave is proportional to the power level.

It would be more elegant if the microwave has a touch screen. However, in this design, we want to keep it as simple as possible, as long as it is straightforward and user-friendly. On top of that, the primary features are well explained without the fancy touch screen's help. Thus, we decide to use a simple LED screen only for displaying the time cook.


User Feedback in the new design.
  • Sound when pressing on the button
  • The button is highlighted when pressed.
  • The higher the power level, the brighter (more yellow) the microwave is
  • Sound when the heating is finished
This new design can help the user avoid abundant steps and focus on the microwave's key features. The knob with some instruction text ("+" and "10" for max, "-" and "1" for min) is intuitive to adjust the power level. The user can adjust both power level and time cook easier and faster. We believe that it is a more user-friendly design for the microwave.
